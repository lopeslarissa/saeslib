# Simplified AES #

[Documentação](https://s-aes.readthedocs.io/en/latest/)

***


## Instalação ##

* ``` $ pip install -r requirements.txt ```
* ``` $ python setup.py sdist ```
* ``` $ pip install -e  s_aes/ ```

***


## Testes ##

* ``` $ tests ```

***


## Criptografar CLI ##

* ``` $ encrypt -k (chave) -t (texto) ```
* ``` $ encrypt -k (chave) -t (texto) -p (opcional: caminho para salvar o relatório) ```

***
